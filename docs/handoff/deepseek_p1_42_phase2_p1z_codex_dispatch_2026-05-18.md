## 3. 内容三段锚点(可任意排序/合并)

### 锚点 A:开锋(GDD §6.5)
- 装备到神物或宝物级,可"开锋"出 3 个槽位
- 每槽可镶嵌一道"锋意"(永久 buff)
- 锋意取自玩家武学领悟、闭关心得、奇缘所得
- 江湖比喻:「剑有锋,人也有锋。器锋利伤敌,人锋利则伤己——所以一柄剑不能只磨一面。」

### 锚点 B:心血结晶(GDD §6.3)
- 装备强化到 +20 之后失败概率渐增,失败时降级
- 心血结晶可保底,使一次强化失败不降级
- 来源:重复武学领悟、节气日闭关副产物、师承遗物拆借
- 江湖比喻:「打铁有句老话:三炼成铜,九炼成钢。每多一锤,都要拿心血换。」

### 锚点 C:心法相生(GDD §4.5)
- 主修 + 辅修心法之间存在 5 种相生组合
- 阴阳调和、刚柔并济、阴影迅捷、同流派精进、同辈互补
- 命中即获得攻速血内多维 buff
- 江湖比喻:「老人讲:水火不容,然水火相济能生药。剑气相违,然剑气相和能生意。」

**建议体例**:不要按"锚点 A/B/C"机械分段;按文学逻辑组织 — 例如:开场点题(到此境界已是江湖少数)→ 三段融合(剑/铁/水火三比喻贯穿,带出三机制)→ 末段收束(到此境界何往)。

## 5. 范例(对齐现有 retreat.md 文风)

```markdown
# 锋·结晶·相生

武学到了一定境界,剑就不只是剑了。神物级的兵器,刃上能开出三道锋——
不是说真的有三道刃口,是说这柄器能容三种锋意。锋意从哪里来?
从一次悟到的招法、一夜闭关的心得、一段无意得来的奇缘里来。
你给它什么,它便回你什么。

强化兵器到了二十重以上,每一次添火都要拿心血换。江湖上有个东西叫
心血结晶,据说是某种至刚至韧的物事——多数人传是大宗师飞升时
散落的内息凝成,也有人说就是一颗血珠,经年累月,在心口凝住了。
有它在,一次失手不至于让前头二十重都白费。

至于心法,练到一辈,人会发现:主修和辅修之间,有时是相克的,
有时却是相和的。两本心法若调子合上了——阴和阳、刚和柔、
快和慢——就生出第三样东西来。这第三样,有时叫"相生",
有时叫"通汇",叫法不一。江湖上的老人讲:"水火不容,然水火
相济能生药。剑气相违,然剑气相和能生意。"到此境界,武学便不再
只是练出来的,是合出来的。
```
