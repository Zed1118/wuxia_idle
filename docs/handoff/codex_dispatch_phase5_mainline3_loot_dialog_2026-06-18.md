## 验收项(逐项截图 + 判 PASS/FAIL)

### A. 关卡列表弹窗(路由 `stage_list`)
1. 进 `stage_list`,任选一关(优先后期关如「黑风岭」之后的高 tier 掉落关),点该关右侧 **ⓘ** 角标。
2. 截「本关传闻」弹窗。**判:**
   - [ ] 水墨 PaperDialog,标题「本关传闻」。
   - [ ] 按桶分组列:桶名(常可得 / 偶可得 / 少有人得 / 江湖传闻 之中出现的),每桶下列物品名;装备带阶色 + 阶名,物品带名。
   - [ ] **不显任何百分比数字 / 不显「传奇·史诗·SSR」等网游稀有词**(红线)。
   - [ ] 若有高于当前境界的掉落物 → 该物标「机缘可遇,火候未到」灰字。
   - [ ] 分组清晰、不溢出、不截断。
3. **点 ⓘ 不应进入战斗 / 不弹进关确认**——只弹传闻浮层(顺带确认)。

### B. 爬塔列表弹窗(路由 `tower_floor_list`)
1. 进 `tower_floor_list`,任选一层点 **ⓘ**。
2. 截弹窗。**判:**
   - [ ] 同 A 的分组列 + 红线(无 % / 无网游词)。
   - [ ] **出现「首通必得」桶**(塔层 dropChance=1.0 项归此桶;A 关卡列表里不应出现此桶)。
   - [ ] **底部脚注「塔层传闻仅首通可得,错过不补」**。
   - [ ] 越阶物「机缘可遇,火候未到」(若有)。

### C. hover tooltip(可选 · 手段受限可跳)
- 鼠标悬停 ⓘ 角标 ~1s,看是否浮出 tooltip「本关传闻」。
- 注:Codex CGEvent 合成 hover 历史上不稳定(参 F-hover 教训),**截不到属验收手段限制非缺陷**,如实标注即可,不影响 A/B 判定。
