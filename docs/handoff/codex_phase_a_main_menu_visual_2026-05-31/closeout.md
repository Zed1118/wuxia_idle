## 窗口尺寸

- 普通窗口最大化：约 1920x974 logical px（macOS 菜单栏 / Dock 限制），截图为 Retina 3840x2160 全屏输出。
- macOS 全屏：1920x1080 logical px，截图为 Retina 3840x2160。
- 1280x720：可用 AppleScript 调到 1280x720 logical px；保留截图已裁成应用窗口区域，实际图片为 Retina 2560x1440。
- 1920 与 1280 下都没有左右空洞或明显遮挡；完整菜单都需要滚动。

## 最影响“像不像游戏”的问题

1. 木牌按钮的材质感偏弱：结构已经像游戏主菜单，但视觉材料仍偏“深色功能卡片”，若后续加木纹、暖色边缘或牌面层次，会更像出版级武侠门面。
2. 菜单纵向过长：首屏能看到标题、修行组和演武开头，但不能一眼看完「修行 / 演武 / 江湖」三组结构；分组成立，但门面的一眼可读性受影响。

## 截图工具链踩坑

- `screencapture -x` 可用，未遇到 Screen Recording 权限阻断。
- Retina 屏幕输出是 2x 像素：1920x1080 logical 会得到 3840x2160 png。
- 普通窗口截图会把桌面其他窗口一起截入；1280x720 图已用 `ffmpeg crop` 裁成应用窗口区域。
- 用 Computer Use 点击「直入江湖」、滚动主菜单；用 `osascript` 读取/设置窗口尺寸与切换全屏。
