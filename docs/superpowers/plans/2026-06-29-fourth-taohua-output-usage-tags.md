# 第 4 梯队：桃花岛产物去向标签

## 目标

- 在桃花岛建筑生产情报中，把已有“产物去向”文字补充为可扫描的短标签。
- 只复用现有 `ItemUsageLookupService` 与配置反查，不新增消耗、产出、售卖或来源聚合逻辑。

## 范围

- UI：`TaohuaIslandScreen` 生产队列信息区。
- 文案：`UiStrings` 集中维护用途到短标签的映射。
- 测试：覆盖源建筑、加工建筑、开锋辅材标签。

## 验收

- 产物用途标签能显示“用于修缮 / 用于加工 / 用于强化 / 用于开锋”等短标签。
- 无产物或无配方时不触发额外消费逻辑。
- 跑桃花岛目标测试与 `flutter analyze`。
