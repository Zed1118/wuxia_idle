# 桃花岛一期 · 设计 spec

> 2026-06-25 brainstorm 收口(用户逐步拍板)。新增「养成经营」支柱:主角隐世基地,种养产料 → 转化战斗力。
> 任务名:**桃花岛一期**。第六阶段战斗轴(流派的形与势 #1/2/4/11/12/16)同期挂账,待本阶段告段落后回。

## 1. 目标与定位

浪迹江湖 → 回岛隐世修行的**生产经营基地**。核心闭环:
`种养产基础材料 → 加工合成 → 喂现有战斗 sink(强化/推境界) → 变强 → 再出江湖`。

用户硬要求:**高自动化 / 零看护负担 / 挂机回收时多条目收获爽感**。

## 2. 红线合规(经营品类天然撞 §5.1/§5.5,焊死如下,违则停)

- ✅ **offline = online**:建筑/配方离线照常推进,关游戏 8h 回来 = 挂 8h 产出。无加速券、无在线 buff(§5.5)。
- ✅ **无体力浇水 / 无每日重置 / 无登录领取 / 无限时作物**(§5.1 体力+每日任务+登录奖励三条)。
- ✅ **仓储 cap 非 FOMO**:沿用闭关 72h 精神防无限堆积,满了停产、不掉、不罚,纯提示回来收。
- ✅ **复用 P4 既有经济**:不新增货币(银两沿用),产出汇入现有 `InventoryItem`/`items.yaml`,不另起平行材料经济。
- ✅ **§5.7 不教程弹窗**:靠解锁 + 剧情自然出现。
- ✅ **数值全 numbers.yaml**:不硬编码。

## 3. 生产模型(自动化增强版 C)

- **原料层(铁匠厂/草药园)= 持续滴落 + 仓储 cap**:每小时产 X,满则停,offline 照算。
- **加工层(打造台/丹房)= 配方队列 + 自动续单**:选定配方后只要原料够就一炉接一炉自动产;原料断暂停、补上自动续。玩家仅在「想换产什么」时点一下,平时零看护。配方离线照常完成。
- **收获爽感**:回岛弹「桃花岛纪事」丰收结算 — 多行明细滚动(精铁 ×N / 药草 ×N / 磨剑石 ×N / 凝神丹 ×N…),数字跳动,强化「一把收」爽感。

## 4. 四建筑机制(两条纵切链)

| 建筑 | 类型 | 产出 | 接入战斗 sink |
|---|---|---|---|
| 铁匠厂 | 原料·滴落 | 精铁(新 miscMaterial) | → 打造台 |
| 草药园 | 原料·滴落 | 药草(新 miscMaterial) | → 丹房 |
| 打造台 | 加工·自动续单 | 精铁→**磨剑石**;高阶(境界+建筑等级锁)→**心血结晶** | → 装备强化 ✅ |
| 丹房 | 加工·自动续单 | 药草→**凝神丹(小)**;高阶→**培元丹(中)** | → 推境界 ✅ |

- 铁链证「产出变强武器」,药链证「产出变消耗」,核心闭环端到端跑通。
- **大还丹(大档经验丹)/ 秘籍仍仅掉落不可产**,守 P4 既有「上货架/掉落」分层设计。
- 一期一种泛用精铁 + 一种泛用药草起步,不铺多品类,二期再扩。

## 5. 建筑升级(产出反哺 · 境界锁)

- 升级提 **产速 + 仓储 cap + 解锁高阶配方**。
- 升级成本 = **银两 + 自产材料**(给银两新 sink + 让产出有去处)。
- **受境界锁(§5.3 精神)**:建筑等级/高阶配方随祖师境界解锁,低境界不能爆产高阶料。

## 6. 数据与工程整合

- 新 `numbers.yaml` 段 `taohua_island`:产速 / 仓储 cap / 配方(投料→成品→时长)/ 升级成本 / 境界解锁门槛。一期保守占位,待 balance 真机校。
- 新 `miscMaterial` 条目:精铁、药草(进 `items.yaml`)。

## 7. 解锁与叙事

- 解锁时机:复用既有 seclusion(闭关)解锁点,或主线第二章末「隐世修行」hook 触发。一期不另搞解锁系统。

## 8. 一期边界(明确不做,留二期+)

木工坊 / 矿洞 / 灵泉 · 闭关&离线收益迁移归口 · 疗伤药&战斗消耗品(撞伤势「无加速疗养」红线,单独想) · 行商码头 · 岛上装饰外观 · 多品类原料。

## 9. 验收口径

- 两条链端到端:种养产料(离线累积)→ 自动续单加工 → 成品入背包 → 可用于强化/推境界。
- offline=online 实测:模拟离线 N 小时,产出 = 在线挂 N 小时(误差 0)。
- 收获结算面板多条目展示。
- analyze 0 / 全量 test 绿 + 新增桃花岛测族 + saveVer 断言全仓同步。
- 红线自查(§2)逐条过。
