## 重要决策

- **决定音频走 profile build 出干净 Steam 截图，因为** VISUAL_ROUTE 调试机制门控在 kDebugMode，release 忽略 route；改 `!kReleaseMode` 后 profile 下 route 生效且 kDebugMode=false 一次隐藏全部 debug chrome（非逐个加 flag）。
- **决定窗口尺寸用 min==max 锁死，因为** Flutter 引擎首帧后会异步 resize 窗口盖过 setFrame，锁 min==max 才钳得住。
- **决定截图 PNG 不入库（gitignore），因为** throwaway，验收结论已记 closeout；仅 manifest + 文本入库。
- **决定音频设置走 shared_preferences 不走 Isar，因为** 设置≠存档，与 Isar save schema 隔离，零迁移风险。
- **决定 T9 报的 3 处 debug「非 bug」不删字段，因为** 已 kDebugMode 门控、release 本不显、对开发有用，只是截图用了 debug build。
