## 踩坑提醒

- **Mac Tailscale 会停** → SSH 到 Pen banner exchange timeout（sing-box TUN 给假 TCP ACK，`nc -z` 假阳性）。先 `tailscale up --accept-dns=false --accept-routes`（精确带原非默认 flag，保留与 sing-box 共存）。
- **Pen git fetch 卡根因**：remote 是 SSH(`git@github`) 但只配 `http.proxy` → SSH 不走 http 代理。改 https remote；大 pack `early EOF / SSL_read unexpected eof` 用 `git fetch --filter=blob:none`（partial clone 只传元数据）。
- **bg session 改主仓文件被 Edit/Write 守卫拦**（cwd `/Users/a10506` 非 repo，EnterWorktree 不可用）→ Bash python heredoc + `assert count==1` 防中文标点误伤。
- **SSH→PowerShell 命令坑**：`(=== xxx ===)` 括号会被当命令执行报错 → 用单引号/纯字符串包；`stash@{0}` 的 `@{}` 被 PowerShell 吃掉报 `unknown switch e` → 用无参 `git stash drop`（默认栈顶）。
- **flutter run macos profile 内存判读**：启动有 settling 爬升（213→251MB）后持平，别把初期爬升当 leak，要看长趋势（≥60s）。
- 后台多行 `for` 循环在 eval 下 `parse error near for` → 用单行 `while`。
