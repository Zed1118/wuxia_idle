## 踩坑提醒

- **bg 写守卫拦 Edit/Write（含 subagent）**：仓库内写一律走 Bash heredoc（`<<'EOF'` 防展开），中文脚本用 `PYTHONUTF8=1 python3 脚本文件`（heredoc 直接 pipe Python 会 non-UTF-8 SyntaxError）。subagent 只能只读 review。别试 EnterWorktree（Isar dylib 截断）。
- **Isar @embedded list 是 fixed-length**：MapLike 的 `add` 在 Isar 取出的 list 上抛「Cannot add to a fixed-length list」，写前必须 `List.of()` 转 growable（`skill_unlock_service.dart` 已修）。
- **改 yaml 后再注入测试易撞重复键**：D1/T3 给 stage/floor 加字段后，红线测试的 brokenLoader 若再 `replaceFirst` 注入同字段 → YAML Duplicate mapping key（非 StateError）。改成**替换**现有值为 ghost id。
- **stage/floor 级字段缩进**：stages.yaml stage 级 = 4 空格（`bossRecruit:`/`name:` 同级），towers.yaml floor 级 = 4 空格（`floorIndex` 后）；注入前 `grep` 确认缩进。
- **commit 前必看 analyze 输出**：本会话 C5 一次性 commit 后才发现 4 个 info 级 lint（prefer_const），项目要求 analyze 0 含 info。每次 commit 前 `flutter analyze <files>` 确认 No issues。
- **balance_simulator 私有 helper 在同文件**：加焦点测要插在 `void main()` 内（锚定 main 闭合括号），别用 `s[:idx]` 截到 EOF——会误删 main 之后的 `_SimResult` 类等顶层函数。
