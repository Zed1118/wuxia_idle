## 踩坑提醒

- **zsh 无引号变量不词分割**：`for b in $list` 把整串当单个词（bash 会分割）。用字面词列表或 `${=list}`。本次清理循环首跑因此失败。
- **merge-tree --write-tree 判冲突**：无冲突时只输出 tree OID；有冲突则 OID 后跟空行 + "Conflicted files" 段。只读预判合并冲突的可靠手段。
- **集成分支识别**：codex 会自建 rollup 集成分支（visual-gap-sweep / integration-dry-run），合并前先 `merge-base --is-ancestor` 查单项分支是否已被包含，避免逐个重复合。
