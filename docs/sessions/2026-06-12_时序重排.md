## 本次完成

把战斗胜利后爆品从"压轴"提为"第一高潮"。纯表现层时序搬移,**0 改 resolve/roll/rng/数值/战斗数学**。

三档时序:
- **爆品档**(掉落含 ≥重器):战斗结束 → flow 播爆品镜头(印章盖落即胜利宣告·轻触继续)→ 结算 dialog
- **普通/无掉落档**:战斗结束 → 简版勝 `VictorySealFlash`(淡入淡出 0.8s 自动·零点击)→ 结算 dialog
- **战败**:VictoryOverlay「敗」不变

实现链(8 task):
1. `BattleStatsSummary`(battle/domain/battle_stats.dart)统计纯函数
2. `BattleScreen.deferVictoryToCaller`:true 时胜利不弹 VictoryOverlay,直接回调下放 flow
3. `VictorySealFlash` + `showVictorySealFlash`(battle/presentation/victory_ceremony.dart)
4. `playTreasureDropIfAny`→bool + `presentVictoryCeremony` 分档
5. 主线 stage_victory_dialog 加统计段
6/7. stage_entry_flow / tower_entry_flow wiring(host defer + presentVictoryCeremony + 统计;塔重打走简版勝)
8. 闸门

## 关键决策 / 发现

- **final review 抓 1 Critical**:`battleProvider` autoDispose,dialog 前现读统计会读到 reset 全零 → 改为在 resolution 函数 `finalState` 有效处算好经返回值带出(commit `6388ee9f`)。
