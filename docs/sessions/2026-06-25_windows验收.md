## 关键过程 / 踩坑

1. **自行定位**：局域网扫 22 端口找到 Windows = `192.168.1.244`（OpenSSH_for_Windows_9.5）。
2. **SSH 通道**：Mac 生成 ed25519 key（`~/.ssh/wuxia_win_acceptance`），用户一次性装公钥到 Windows `administrators_authorized_keys`（唯一人工步）。用户名 `administrator`。**Windows 在线即可复用此通道。**
3. **GitHub clone 失败**：`RPC failed; SSL_read unexpected eof / early EOF`——这台 Windows 到 GitHub 大传输不稳，浅克隆也失败。改 **Mac→Windows LAN 直传**。
4. **scp 须加 `-O`**：默认 SFTP 协议在该 Windows sshd 不通，legacy `-O` 才行。
5. **Isar 无需手拷**：Windows 端 `flutter test` 自带 libisar（isar_community_flutter_libs），不像 Mac 要手动 cp libisar.dylib。
6. **detached 跑 build 不可靠**：`start /b` / Start-Process 重定向日志反复失败（flutter.bat 交棒 dart 后重定向断）；**同步 SSH + ServerAliveInterval 保活** 才稳（build 94.8s 在 600s timeout 内）。
7. **assets 必传**：只传源码时 10 个「资源存在性」测试假失败（audio_assets/asset_audit），补 assets（300M，scp -O，字节校验一致）后清零。
