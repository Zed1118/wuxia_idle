### Step 3: PvpScreen(`lib/features/pvp/presentation/pvp_screen.dart` 新建)

`ConsumerWidget` 三态(沿 `light_foot_screen.dart` 体例):
- **locked**:stage_05_05 未通(读 `mainlineProgressProvider.clearedStageIds`)→ 显锁定文案 + 「主线 Ch5 通关后解锁」hint
- **available**:显当前 ELO + RankBadgeWidget + match button(onPressed → PvpService.match)+ PvpHistoryList
- **cleared**:PVP 无终态(始终 available)

```dart
class PvpScreen extends ConsumerWidget {
  const PvpScreen({super.key});
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    // 读 clearedStageIds 判 unlock
    final progress = ref.watch(mainlineProgressProvider);
    final unlocked = progress.value?.clearedStageIds.contains('stage_05_05') ?? false;
    if (!unlocked) return Scaffold(appBar: AppBar(title: const Text('论剑对决')), body: const Center(child: Text('主线 Ch5 通关后解锁')));
    // available 状态
    return Scaffold(...);
  }
}
```

> 注:`mainlineProgressProvider` 真名按 Phase 0 grep 实际为准(可能叫 `clearedStagesProvider` 等)。
