## 目标

### 1. 加 ReputationService helper(`lib/features/jianghu/application/reputation_service.dart`)

加 `deltaApplierFromRng(Random rng) → ReputationDeltaApplier`:
```dart
ReputationDeltaApplier deltaApplierFromRng(Random rng) {
  return ({required playerId, required factionId, required deltaMin, required deltaMax}) async {
    final delta = deltaMin + (deltaMax >= deltaMin ? rng.nextInt(deltaMax - deltaMin + 1) : 0);
    await applyDelta(playerId: playerId, factionId: factionId, delta: delta);
  };
}
```

理由:闭包构造逻辑统一在 service · 两个 caller 复用 · test 端易 fake。
