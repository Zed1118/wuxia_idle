### Phase 1 · 写 4 个 yaml 文件(35-45 min)

#### 1. `data/narratives/techniques/bing_pian_xin_jue.yaml` · 冰魄心诀(yinRou)

origin 题材:北地寒山道观 · 一位无名女道长留下的内功,练成者周身冰凉,但目光暖。

3 招建议:
- `bing_pian_ru_jing`(冰魄入境):静坐三时,周身寒气内敛,百脉自通
- `bing_pian_an_du`(冰魄暗渡):一掌击出无声无息,被击者三日后才察觉脏腑发凉
- `bing_pian_xue_chen`(冰魄雪沉):内劲化作寒冰般沉重,落地一掌可让对方脚下生根

加 `mantra:` 短偈(4 行),贴 yinRou 阴寒内功的克制感。

#### 2. `data/narratives/techniques/chi_yang_jin_gang_quan.yaml` · 赤阳金刚拳(gangMeng)

origin 题材:西北某寺 · 老和尚一夏伐木百日,把伐木的劲化为拳法。每招打出如金石相击,有炽热感但不烧人。

3 招建议:
- `chi_yang_pi_shan`(赤阳劈山):一拳直下,劲力如阳光垂直洒落,可破石不伤土
- `chi_yang_xi_fang`(赤阳熄锋):接对方兵刃,以拳面磕击使刀剑铿然作响,内劲渗入兵刃使其暂时失锋
- `chi_yang_man_yin`(赤阳满印):全身贴近对方,拳劲透三层布,留下一个温热的掌印 — 可拳印不烧人

不加 mantra(外功拳法,体例参考 `po_jun_quan.yaml` 无 mantra)。

#### 3. `data/narratives/techniques/liu_yun_qing_ling_shen_fa.yaml` · 流云轻灵身法(lingQiao)

origin 题材:川中某剑客自创身法 · 闲时看云,看云朵从一山飘到另一山,把云的节奏融入身法。

3 招建议:
- `liu_yun_du_jiang`(流云渡江):一步跨过三丈,落地无声,踩在水面上荡起一圈涟漪即收
- `liu_yun_fan_pian`(流云翻篇):身形侧转九十度,如风过书页,只留下风,不留下人
- `liu_yun_gui_yan`(流云归雁):疾跃之后忽然回身,身法划出一道弧,如雁折翅归来

加 `mantra:`(身法清诀,4 行短偈)。

#### 4. `data/narratives/techniques/tai_yi_xuan_shen_jue.yaml` · 太一玄神诀(中性高阶内功)

origin 题材:终南山某道观失传的内功秘籍 · 自周朝传下,只在道藏夹页中残留 9 个字。后世道人各自演绎,谁也不敢说自己练的是「真」太一玄神诀。

3 招建议:
- `tai_yi_xuan_xi`(太一玄息):一念静坐,内息回流至脐下三寸,如有山林雾气在丹田打转
- `tai_yi_gui_yi`(太一归一):内劲收敛至极致,周身气息若有若无,敌人难以察觉气场
- `tai_yi_fan_pu`(太一返朴):内息回到最初的一缕,身轻如初学者 — 但这一刻所有招式都从头处生

加 `mantra:`(道家偈,4 行)。

### 严格遵守

- **沿既有体例**:id / name / origin: | / [可选 mantra: |] / moves: list(3 招,每招 id+name+description: |)
- **每招 description 3-5 行**,武学要旨 + 意境留白
- **不写战斗效果数值**(伤害多少 / 概率多少 / cooldown)
- **三流派语言区分**:gangMeng 写拳掌震 / lingQiao 写步法身形 / yinRou 写内息阴寒
- **黑名单词全无**:legendary / epic / 史诗 / 神器 / 传说级 / 无敌 / 最强 / 究极 / 霸气 / 逆天 / 刀光剑影 / 血溅
- **0 emoji**
- **文件长度 50-100 行 per file**(400-700 字)
- **id 字段与文件名一致**
