3. **lib/data/defs/ def 字段使用率**(memory `feedback_extension_hardcode_audit` 重点):
   ```bash
   ls lib/data/defs/ 2>/dev/null
   ```
   对每个 `*_def.dart`:
   - grep class field 列表(`final \w+ \w+;` 或 `final \w+ \w+,` in constructor)
   - 对每个 field name,grep 在 lib/ 内的引用次数
