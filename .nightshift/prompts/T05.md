## 2. 真解 · NavigatorObserver Mock + 单帧 pump

```dart
class _RecordingNavigatorObserver extends NavigatorObserver {
  final pushedRoutes = <Route<dynamic>>[];
  @override
  void didPush(Route<dynamic> route, Route<dynamic>? previousRoute) {
    pushedRoutes.add(route);
    super.didPush(route, previousRoute);
  }
}

testWidgets('tap X 按钮 → Navigator.push 触发', (tester) async {
  final observer = _RecordingNavigatorObserver();
  await tester.pumpWidget(
    ProviderScope(
      child: MaterialApp(
        navigatorObservers: [observer],
        home: const MainMenu(),
      ),
    ),
  );
  expect(observer.pushedRoutes.length, 1); // initial route
  await tester.tap(find.text('按钮标签'));
  await tester.pump(); // 单帧不 settle
  expect(observer.pushedRoutes.length, 2);
  expect(observer.pushedRoutes.last, isA<MaterialPageRoute<void>>());
});
```

## 3. 套路核心

(三段)
- **不验证子屏内容,只验 push 触发**(降验收范围 = 降风险)

- **NavigatorObserver 是 Flutter 标准 API**,不引入新 mock 库(无 deps 成本)
