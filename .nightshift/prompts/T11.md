### Step 2: PvpRecord @Collection(`lib/features/pvp/domain/pvp_record.dart` 新建)

沿 spec §2 体例。完整 imports + Isar Collection。composite index 写 `(playerId, timestamp)` 沿 P1.2 enmity / P2.x inner_demon 体例。

```dart
import 'package:isar/isar.dart';

part 'pvp_record.g.dart';

@Collection()
class PvpRecord {
  Id id = Isar.autoIncrement;
  late String matchId;                  // uuid v4(application 层生成)
  @Index(composite: [CompositeIndex('timestamp')])
  late int playerId;                    // 本地玩家 character id
  late int opponentSnapshotId;          // → PvpSnapshot.id
  late int leftSnapshotId;              // 玩家阵容快照
  late int? winnerId;                   // null = draw
  late int playerEloBefore;
  late int playerEloAfter;
  late int eloDelta;                    // ±单场积分变化
  late DateTime timestamp;
}
```
