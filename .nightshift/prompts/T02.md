## 背景(无需展开 lookup)

挂机武侠 Flutter Desktop 武侠 idle 游戏。P1 #42 Phase 1(2026-05-17)新落 `HomeFeedScreen`(`lib/features/home_feed/presentation/home_feed_screen.dart` 193 行)含**快速领取按钮**:
- 点击触发 `markAllFeedRead` 顶级函数(把 unread feed 全 mark 为 isRead=true)
- 然后 `Navigator.pushReplacement` 到主菜单 / 战斗屏(项目实际目标见 lib/ 代码)

现有 `home_feed_screen_test.dart` 仅测了「按钮 visible」(`find.text('快速领取')` 之类),**未测 tap 行为 / 重复点 / 空 list 容错**。本任务补这 3 个边界 widget test,产出独立 `_quick_claim_test.dart` 不动现有 test。

## 红线:约束语义不写瞬时事实

写法约束:用 `find.byType(NavigatorObserver)` mock 监 push 是否触发(非次数);用 `verifyNever` / `verify(...).called(anyNamed)` 体例;集合断言用类型/语义而非"3 次"。

参考 `feedback_e2e_playwright_pitfalls`(用户全局 memory)对 mock pattern 的提示。
